# Autonomous AI Agents for Code Repository Management

This workspace contains a small Python sample project for testing repository tooling and GitHub extension workflows.

## Files

- `addition.py` - adds two numbers
- `subtraction.py` - subtracts two numbers
- `multiplication.py` - multiplies two numbers
- `division.py` - divides two numbers
- `main.py` - runs a quick demo
- `test_calculator.py` - basic tests

## Run

```bash
python main.py
python -m unittest test_calculator.py
```
